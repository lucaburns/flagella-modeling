# Elastic Rod Modeling of Flagella

This project studies the equilibrium behavior of flagella in single-cell organisms using the Kirchhoff rod model and the immersed boundary (IB) method. The goal is to derive explicit analytical expressions for the force and torque generated by elastic filaments with intrinsic curvature and twist.

Building on prior work by Lim, which focuses on circular filaments, this project extends the framework to more general geometries using helical parameterizations. Explicit solutions are obtained for both the stretching of the rod and the orientation of the material frame, enabling analysis of multiple biologically relevant flagellar shapes.

The derivations are implemented symbolically and used to compute equilibrium configurations where the net force and torque vanish. These solutions provide insight into how filament geometry influences mechanical behavior and can serve as benchmarks for computational simulations.

## Summary of Results

* Derived explicit expressions for force and torque in equilibrium configurations
* Solved for key parameters governing:

  * rod stretching
  * triad orientation
* Extended circular filament results to multiple helical geometries, including:

  * constant pitch helices
  * parabolic helices
  * variable-radius helices
* Demonstrated that equilibrium solutions can be computed analytically for these shapes

These results provide a foundation for analyzing more complex filament dynamics and for validating numerical models.
